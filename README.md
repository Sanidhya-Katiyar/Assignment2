AID Transfer Learning Study

This repository contains experiments for evaluating transfer learning performance of CNN backbones on the AID remote sensing dataset.

Models:
- ResNet50
- EfficientNet-B0
- ConvNeXt-Tiny

Experiments:
- Linear probing
- Fine-tuning strategies
- Few-shot learning
- Corruption robustness
- Layer-wise probing

Dataset:
AID Dataset
http://captain.whu.edu.cn/AID/

Dataset should be placed in:

data/AID/
